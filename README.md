# Path to Success — HTML Template

This repository contains a minimal, accessible HTML starter template.

Files added/modified:
- `Index.html` — main HTML template (responsive, accessible, semantic)
- `css/styles.css` — minimal stylesheet for layout and components
- `js/script.js` — small script for menu toggle and runtime behaviors

How to use
1. Open `Index.html` in your browser (double-click or use a local server).
2. Edit content directly in `Index.html`.
3. Customize styles in `css/styles.css` and behaviors in `js/script.js`.

Notes
- The template uses a mobile-first responsive design and includes a skip link for accessibility.
- The contact form is non-functional by default (`onsubmit="return false;"`). Hook it to a backend or form service when ready.
